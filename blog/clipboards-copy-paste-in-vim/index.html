Clipboard's copy-paste in Vim
ad
Mar 11, 2018
One minute read

Simple “copy-paste” method from clipboard using xsel

Install xsel

example using arch :

sudo pacman -S xsel

Create some keymap on your ~/.vimrc

" Copy to X CLIPBOARD
map <leader>cc :w !xsel -i -b<CR>
map <leader>cp :w !xsel -i -p<CR>
map <leader>cs :w !xsel -i -s<CR>
" Paste from X CLIPBOARD
map <leader>pp :r!xsel -p<CR>
map <leader>ps :r!xsel -s<CR>
map <leader>pb :r!xsel -b<CR>

note that <leader> here is mapped to backslash key \ by default
